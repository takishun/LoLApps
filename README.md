# LoL 装備金額クイズアプリ

League of Legends の装備の金額を合計するクイズアプリです。Streamlit で構築されています。

## 機能

- ランダムに選ばれたLoL装備の合計金額を計算するクイズ
- 3つの難易度レベル（簡単・普通・難しい）
- スコアと連続正解数の追跡
- 正答率の統計表示
- 答えを見る機能
- アクセスカウンター（訪問者数を表示）

## セットアップ

### 必要要件

- Python 3.8 以上

### インストール

1. リポジトリをクローン:
```bash
git clone <repository-url>
cd LoLApps
```

2. 依存関係をインストール:
```bash
pip install -r requirements.txt
```

### 実行方法

以下のコマンドでアプリを起動:

```bash
streamlit run app.py
```

ブラウザで自動的に開かれます（通常は http://localhost:8501）

## 使い方

1. 難易度を選択（サイドバー）
2. 表示された装備の合計金額を計算
3. 金額を入力して「回答する」ボタンをクリック
4. 正解・不正解が表示されます
5. 「次の問題」で新しいクイズに進みます

## 難易度

- **簡単**: 2-3個の装備
- **普通**: 3-4個の装備
- **難しい**: 4-6個の装備
